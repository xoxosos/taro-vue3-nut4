# taro-vue3-nut4
使用taro-nutui4+vue3+ts+pinia+husky搭建的通用小程序,H5模版,开箱即用.
## 一.项目安装
```bash
# 克隆项目 https
 git clone https://github.com/taro-vue3-nut4.git
# 克隆项目 ssh
 git clone git@github.com:xoxosos/taro-vue3-nut4.git
# 进入项目目录
cd taro-vue3-nut4
# 安装依赖
 npm i
# 推荐使用pnpm 安装pnpm
 npm i pnpm -g
# pnpm安装依赖
 pnpm i 
 
```
